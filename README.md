# 海创元 2026 年度组织绩效管理系统

版本：V3.1 · 单文件静态 HTML · 无需后端

## 快速开始

### 本地使用

直接双击 `index.html`（等同于 `海创元2026年度组织绩效管理系统_V3.html`）即可在浏览器中打开。

### Replit 部署

1. 在 Replit 中 Import from GitHub，选择本仓库
2. 模板选择 **HTML/CSS/JS** 或 **Static**
3. Run 后即可通过 `xxx.replit.app` 访问
4. 若需要自定义启动命令：`python -m http.server 8000`

### 其他静态服务器

- **Nginx**：将仓库文件夹放入 `/var/www/html/`，访问 `/` 即默认加载 `index.html`
- **Python**：`cd` 到目录后执行 `python -m http.server 8000`
- **Node.js**：`npx serve .`

## 自定义与维护

- **修改账号密码**：编辑 HTML 顶部 `const ACCOUNTS = [...]` 数组，修改后保存即生效
- **更新财报月度值**：编辑 `const FIN_MONTHLY = {...}`，递增 `FIN_REPORT_VER` 版本号触发重新注入
- **清空本地填报**：用户端 F12 控制台执行 `localStorage.clear(); location.reload()`

## 技术栈
